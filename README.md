# ⚡ FORGE STUDIO — AI Motion Tracking Engine

Forge Studio is a high-performance fitness tracking application that uses **Computer Vision** and **Machine Learning** to provide real-time rep counting and form analysis. Designed for athletes who want precision data without manual logging.

---

## 🚀 Key Features
* **Real-Time Rep Counter:** Automatically detects movements for 40+ exercises.
* **AI Form Analysis:** Skeleton mapping to identify and correct posture.
* **Live Telemetry:** HUD-style visualization of power, velocity, and time-under-tension.
